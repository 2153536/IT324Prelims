Hypertext Markup Language (HTML)

    Definition

      • Language used to markup documents in the World Wide Web (WWW)
      • Used to specify structure and content, as well as presentation of web pages
      • Just for semantic specification of web page structure and content in modern web development
      • Initially developed at CERN, then IETF, then W3C, then WHATWG

    HTML 4.01

    • Strict Transitional, Frameset Document Type Definitions (DTDs)
    • Introduced depracation of presentational elements and attributes in favor of style sheets

    Extended Hypertext Markup Language (XHTML)

    • Reformulation of HTML in XML
    • Intended to facilitate the intro of new HTML elements and/or attributes
    • Provide interoperability between HTML and HTML-based systems
    • Leverage the use of Extended Markup Language (XML) tools in creating, editing and parsing HTML
    • Sought to move the Internet content-development community from the days of malformed standard HTML markup into the well-formed valid world of XML

    HTML 5

    • Arose from the effort to evolve HTML instead of replacing it
    • Address backward compatibility issues that hindered the adoption of XHTML
    • Address the emerging needs of modern web applications
    • Incorporated specifications from HTML4, XHTML and Document Object Model (DOM)
    • Included various extensions of existing Application Programming Interfaces (APIs) and introduced new APIs

    Version History

    • Late 1991 - HTML Tags by Time Berners-Lee
    • Mid 1993 - HTML Internet Draft by Tim Berners-Lee and Dan Connolly published by IEFI
    • Late 1993 - HTML Internet Draft by Dave Raggett
    • Nov 1995 - HTML 2.0 (RFC 1866)
      • Supplemental RFCs: RFC 1867, RFC 1942, RFC 1980, RFC 2070
    • Jan 1997 - HTML 3.2 (W3C Recommendation)
    • Dec 1997 - HTML 4.0 (W3C Recommendation)
    • Dec 1999 - HTML 4.01 (W3C Recommendation)
    • Jan 2000 - XHTML 1.0 (W3C Recommendation)
    • May 2001 - XHTML 1.1, Module-based XHTML (W3C Recommendation)
    • 2004 - Work on HTML was initially undertaken by Web Hypertext Application Technology Working Group (WHATWG) led by Apple, Mozilla and Opera
    • 2006 - W3C signifies the interest to participate in the effort of making HTML 5
    • 2008 - The first draft of HTML5 was published. It is edited by Ian Hickson.
    • Oct 2014 - HTML 5 (W3C Recommendation)
    • Nov 2016 - HTML 5.1 (W3C Recommendation)
    • Dec 2017 - HTML 5.2 (W3C Recommendation)
